Summary: Latitude vs. Weather Conditions

This is an analysis of 500+ cities across the world to determine if proximity to the equator has an affect on weather conditions. Data was compiled from the OpenWeatherMapAPI for this project.

The API pull from OpenWeatherMap provided coordinates for each city as well as temperature, cloud coverage, humidity, and windspeed. This site utilizes charts from Matplotlib to compare Latitude, our metric for proximity to the equator, to the various weather conditions. You will also find the source data for this project with the weather comparisons.
